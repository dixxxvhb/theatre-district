## Session 2 — The Marquee Noir rendering kit (2026-06-12)

The look of the game, built as a kit every later session draws from (`src/game/render/kit/`). **Palette** (`palette.ts`): ink/tungsten/neon/surface/facade tokens plus the time-of-day grade — six keyframes lerped across the day (flat afternoon → golden hour → curtain night → wind-down), expressed as ONE world-container tint + sky tints + a light-layer master alpha; zero per-sprite work. **Bake pipeline** (`bake.ts`): all procedural Graphics bake to textures once at boot, cached by key; radial glow and canvas-noise helpers. **Building factory** (`buildings.ts`): iso shells with facade detail on the visible SW face, each variant baked as a base + emissive PAIR — base darkens with the grade in the world container, emissive sits on the additive light layer so windows/marquees ignite as night falls (identical-bounds frames keep the pair pixel-aligned). First variants: the derelict playhouse (boarded windows, dead marquee, grime streaks, mostly-dead bulb row) and its restored sibling — both carry the memorial mark, two outlined gold circles by the stage door, quiet and permanent. **Atmosphere** (`atmosphere.ts`): two-sprite tint-only sky with the horizon glow hugging the top of frame, tiling film grain (static under `prefers-reduced-motion`), pooled steam wisps off sidewalk grates (world-space so the grade tints them). **StreetScene** replaces the debug view: tile variants picked by deterministic hash (asphalt with center-line dashes, slab-seamed sidewalks, parcel-lined lots), era-aware lamp dressing (Dark Block: most lamps dead, one hero lamp by the playhouse always flickering), bloom as a single BlurFilter on the additive light container. `newGame` now seeds the inherited derelict playhouse mid-block north. Dev panel gained Noon/Dusk/Night/Late time jumps (lighting can't be playtested at 90s/day). Fixed the inherited favicon-404 console error (inline SVG marquee-bulb icon). 33/33 tests green, build clean, 60fps, zero console errors.
